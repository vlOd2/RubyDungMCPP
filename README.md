# RubyDungMCPP
RubyDung port to C++/CLI, compatible with Windows XP, see requirements below

# Requirements
- (XP ONLY) [.NETFX 4.0](https://www.microsoft.com/en-us/download/details.aspx?id=17718)
- [Visual C++ 2015 Runtime](https://www.microsoft.com/en-us/download/details.aspx?id=48145)

# Build requirements
- Visual studio 2019 or higher (older versions probably work too)
- .NET, C++ and C++/CLI support
- [.NETFX 4.0 SDK](https://gist.github.com/vlOd2/1c3c15ac559c30b90cde709d84b39e60)
- C++ v141_xp build tools (make sure to get the C++/CLI ones too)
